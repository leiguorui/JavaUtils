# 欢迎使用 JavaUtils

---

JavaUtils 是一套Java编程中常用的工具类,具体如下:

> * DateUtils
> * PageModel
> * 简单的分词器 (复杂的分词器见：https://github.com/leiguorui/ansj_seg)

#### 安全验证、加密

> * Md5Utils
> * CryptoFile 文件的加密、解密

#### 缓存、内存数据库的读写

> * MemcachedUtil
> * redis的读取，和序列化

#### 文件操作

> * poi 读写excel
> * 文件路径的读取
> * 单例读取properties配置文件
